# 🎬 MovieFlix

A modern movie discovery web application built with **React**, **Vite**, **Tailwind CSS**, and **CSS**.  
The application fetches the latest movie data from the **TMDB API** and displays detailed information in a clean and responsive interface. Users can also search for any movie by name.

> **Note:** Since TMDB is restricted in some regions, you need to use a **VPN** for the application to work properly.

---

## 🚀 Features

- 🎥 Display the latest movies from TMDB
- 🔍 Search movies by title
- ⭐ Show movie ratings
- 📅 Display release year
- 🖼️ Show movie posters
- 📱 Responsive design for desktop and mobile
- ⚡ Fast development with Vite

## 📋 Prerequisites

Before running this project, make sure you have installed:

- Node.js
- npm (comes with Node.js)

You can verify the installation by running:

```bash
node -v
npm -v
```

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Go to the project folder:

```bash
cd your-repository
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the project root and add your TMDB API key:

```env
VITE_TMDB_API_KEY=YOUR_API_KEY
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## 🌐 API

This project uses data from **The Movie Database (TMDB) API** to fetch:

- Latest movies
- Movie posters
- Ratings
- Release dates
- Search results

---

## ⚠️ Important

TMDB services may be blocked in some countries.

**Please use a VPN before running the application**, otherwise the movies may not load.
